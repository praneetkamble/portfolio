# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/praneetkamble/portfolio.git
```

Navigate into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
